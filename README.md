# Comparative-Analysis-NN
This project compares CNNs and fully connected neural networks (DNNs) for image classification, exploring optimal configurations and trade-offs across diverse datasets.

This research was conducted during Summer 2023 and compares fully connected neural networks (DNNs) and convolutional neural networks (CNNs) for image classification across MNIST, EMNIST, Fashion MNIST, and CIFAR-10. The project analyzes how architecture choice, training duration, and model size affect classification accuracy, and includes visualizations summarizing performance trade-offs across datasets.
